## Final Version: Online Web Tool (NEW!)

I have created a modern web interface for you! You can now just drag and drop your PDFs into a browser and get the merged file instantly.

### How to Run the Site:
1. Ensure your virtual environment is active.
2. Run the Flask app:
   ```powershell
   venv\Scripts\activate
   python app.py
   ```
3. Open your browser to: **http://127.0.0.1:5000**
4. Drag your PDFs into the purple box and click "Process & Download".

## Features
- **4 Index Layout**: Top-Left, Top-Right, Bottom-Left, Bottom-Right.
- **Centered Placement**: Every label is perfectly centered in its index.
- **Dashed Cutting Guides**: Gray dashed lines to help you cut the page.
- **No Size Change**: Labels are kept at their original size (no scaling).
- **Modern UI**: Clean, purple-themed web interface.
